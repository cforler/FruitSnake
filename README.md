# Fruit Snake

I beg you snake, eat the red fruits, not yourself, please. ;-)


![Screenshot](screenshot.png)


## Build an run the application (jar file)
```shell
$ make 
```

## Run the application (jar-file)
```shell
$ make run
```

Alternativ approach
```shell
$ java  -jar FruitSnake.jar
```
